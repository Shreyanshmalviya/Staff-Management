# Staff Record Management Program using C

*Made alongside* [@KirtimaanSingh](https://github.com/KirtimaanSingh)

A mini-project made in C for staff record management with following commands:
* Add staff record
* Delete staff record
* Modify staff record
* Display staff record
* Exit

Libraries used:
* stdio.h
* stdlib.h
* string.h
* windows.h
